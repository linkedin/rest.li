# Rest.li github pages 

## How to contribute

Create a local branch:

```
git clone 
git checkout -b [your_branch]
```

Once you are done with your changes, you can push them:

```
git add -A
git commit -m 'your comment'
git push origin [your_branch]
```

Create a PullRequest
